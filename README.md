Sales Data Analysis using Superstore Dataset

->Problem Statement

The objective of this project is to analyze sales and profit data from a retail business to identify trends, top-performing products, and business insights that can support decision-making.

->Dataset Details

* Dataset: Superstore Sales Dataset
* Source: Kaggle
* Features include order information, sales, profit, customer details, product categories, and regional data.

->Approach

1. Loaded the dataset using Pandas.
2. Converted the Order Date column into datetime format.
3. Performed exploratory data analysis.
4. Analyzed monthly revenue trends.
5. Identified top-selling products and sub-categories.
6. Conducted profit analysis across categories.
7. Created visualizations using Matplotlib.

->Results

* Identified monthly sales trends.
* Determined top-performing product categories and sub-categories.
* Compared profit contributions across categories.
* Generated visual insights to support business decision-making.

->Technologies Used

* Python
* Pandas
* Matplotlib
* Google Colab / Jupyter Notebook

->Conclusion

The analysis provided valuable insights into sales performance and profitability. Such insights can help businesses optimize inventory, marketing strategies, and overall operations.
